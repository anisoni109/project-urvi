# Project Urvi Website

A modern one-page website concept for Project Urvi, centered on a sustainable plastic recycling plant and supported by research-led sustainability services.

## Files

- `index.html` - page structure
- `styles.css` - layout and visual design
- `script.js` - mobile navigation and reveal animations

## Use

Open `index.html` in a browser to preview the website.
